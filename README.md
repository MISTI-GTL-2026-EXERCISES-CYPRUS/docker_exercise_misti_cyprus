# Docker_exercise
Repository with tasks, that will help students to understand the principals of docker and how to use it further

## Exercises
1. [01-running-containers](01-running-containers/README.md) - Learn how to run existing Docker images.
2. [02-building-images](02-building-images/README.md) - Learn how to build images using a provided Dockerfile.
3. [03-dockerfile-from-scratch](03-dockerfile-from-scratch/README.md) - Create your own Dockerfile for a Node.js application.
